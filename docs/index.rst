Welcome to the NexusRT documentation. NexusRT is a firmware-centric,
OS-bypass runtime for end-to-end LLM pipelines on NVIDIA CUDA GPUs and
supported Apple Silicon Metal devices. A100/H100 are the primary feature
targets; T4/P100-class Kaggle GPUs are used for CUDA smoke validation. See the
repository README for the current venv, CMake, Python install, Kaggle, and
smoke-test workflow.
